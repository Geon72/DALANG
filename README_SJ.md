* 금융감독원 정기예금 API를 활용하여 예금 상품 서버에 저장 완료!
  * 필요한 데이터만 선택함
    * kor_co_nm : 은행명
    * fin_prdt_nm : 상품명
    * intr_rate_type : 저축 금리 유형 / 이자 계산방식
    * save_trm : 저축 기간 [단위: 개월]
    * intr_rate : 일반 금리
    * intr_rate2 : 우대 금리
    * join_way : 가입 방법
    * spcl_cnd : 특별 조건
    * join_deny : 가입 제한 가입제한 Ex) 1:제한없음, 2:서민전용, 3:일부제한
    * join_member : 가입 대상
    * etc_note : 기타 사항
    * created_at : 생성 날짜
  * 추후 금융감독원 정기적금 API를 활용하여 적금 상품도 서버에 저장 해야 함!!!(20241119 예정)
